# Version 2.0

Esta versión introduce un cambio arquitectónico con respecto a la versión 1.0 para una mayor flexibilidad y facilidad de uso.

*   **Nuevo Motor de Renderizado (mPDF + Smarty):** Se migró de TCPDF a mPDF integrado con motor de plantillas Smarty.
*   **Plantillas Dinámicas:** Ahora se utilizan archivos `.tpl` (HTML + Smarty) y CSS. Esto permite manipular la estructura del PDF como si fuera una página web, simplificando enormemente el diseño.
*   **Sistema de Catálogo:** La estructura del PDF se define declarativamente en un archivo `catalog.xml`, desacoplando la configuración visual de la lógica PHP.
*   **Facilidad de Edición:** Ya no es necesario programar clases PHP para cada parte del PDF; basta con editar los archivos de la plantilla (HTML/CSS).
