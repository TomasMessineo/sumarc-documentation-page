# Versión 1.0

**Lanzamiento Inicial**
*   **Adaptación SciELO**: Transformación de XML JATS estándar al perfil específico de SciELO.
*   **Validación de Estructura**: Verificación de cumplimiento con DTDs y reglas de marcado SciELO.
*   **Ajuste de Metadatos**: Normalización de afiliaciones, fechas y referencias según requisitos de la indexadora.
*   **Integración**: Funciona sobre archivos previamente procesados por SUMARC.
