# Versión 1.1

## Resumen
Versión extendida por SEDICI con soporte para metadatos personalizados y nuevos roles de contribuyentes.

## Novedades

### Referencias Bibliográficas
- **Nuevos Person-Groups**: Soporte para roles adicionales como `coordinators`, `compilers`, `translators`.
- **Nuevos Campos**:
    - Agregado soporte para `degree` (Grado), `awardingInstitution` (Institución) y `publicationNumber` en referencias de tipo Tesis.

### Importación/Exportación
- Mapeo bidireccional completo entre los nuevos campos visuales y el XML JATS estándar.
