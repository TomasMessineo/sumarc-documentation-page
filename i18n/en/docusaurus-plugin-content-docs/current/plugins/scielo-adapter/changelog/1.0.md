# Version 1.0

**Initial Release**
*   **SciELO Adaptation**: Transformation of standard JATS XML to the specific SciELO profile.
*   **Structure Validation**: Verification of compliance with SciELO DTDs and markup rules.
*   **Metadata Adjustment**: Normalization of affiliations, dates, and references according to indexer requirements.
*   **Integration**: Works on files previously processed by SUMARC.
