# Version 1.1

## Summary
Extended version by SEDICI supporting custom metadata and new contributor roles.

## New Features

### Bibliographic References
- **New Person-Groups**: Support for additional roles such as `coordinators`, `compilers`, `translators`.
- **New Fields**:
    - Added support for `degree`, `awardingInstitution`, and `publicationNumber` in Thesis references.

### Import/Export
- Full bidirectional mapping between the new visual fields and standard JATS XML.
