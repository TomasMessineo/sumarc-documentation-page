# Versión 1.0

**Lanzamiento Inicial con Soporte APA 7**
Esta versión incorpora una mejora fundamental sobre el plugin original: la capacidad de identificar y estructurar referencias bibliográficas automáticamente.

## Funcionalidades Clave

*   **Conversión DOCX a JATS**: Funcionalidad base de conversión de estructura, párrafos, tablas y figuras.
*   **Identificación de Referencias (APA 7)**:
    *   Implementación de un nuevo submódulo `citation-parser` basado en expresiones regulares.
    *   Capacidad de extraer autores, año, título, fuente y DOI/URL de referencias en texto plano formateadas en APA 7.
    *   Generación automática de etiquetas `<ref-list>` y `<element-citation>` estructuradas, reemplazando las citas mixtas (`<mixed-citation>`) del desarrollo original.
*   **Integración con docxToJats**: El parser de citas se inyecta en el flujo de conversión de `docxToJats`, procesando la sección de bibliografía detectada en el documento.
