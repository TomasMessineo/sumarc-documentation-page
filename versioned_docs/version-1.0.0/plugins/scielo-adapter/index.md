# scieloAdapter

**scieloAdapter** es un plugin de especialización para el ecosistema SciELO.

## Función
Adapta un archivo JATS XML estándar (generado por SUMARC) para cumplir estrictamente con las especificaciones y validaciones requeridas por la metodología SciELO (Scientific Electronic Library Online).

## Características
*   Realiza transformaciones específicas en la estructura del XML.
*   Valida el cumplimiento de las DTDs o esquemas específicos de SciELO.
*   Asegura que el documento sea aceptado por el sistema de publicación de SciELO.

## Uso en el Flujo
Es un paso opcional pero crucial si el destino de la publicación es una revista indexada en SciELO. Se utiliza sobre el XML ya enriquecido antes de la entrega final.
