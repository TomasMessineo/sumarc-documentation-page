# Versión 1.0

**Lanzamiento Inicial**
*   **Extracción de Metadatos**: Integración con la API de OJS para obtener datos de envío, publicación y contexto.
*   **Generación de Front**: Construcción automática de la sección `<front>` conforme a JATS 1.2/1.3 y perfiles SciELO.
*   **Previsualización**: Interfaz para revisar los cambios en metadatos antes de aplicarlos.
*   **Integración en Flujo**: Pestaña dedicada en la etapa de Producción de OJS.
