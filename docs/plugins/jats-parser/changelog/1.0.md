# Version 1.0

*   **Motor TCPDF:** Se utiliza la librería TCPDF para la generación de documentos.
*   **Patrón Strategy:** La selección de diseños se maneja mediante clases PHP con el patrón Strategy.
*   **Renderers Programáticos:** Los elementos del PDF (títulos, autores) se imprimen mediante clases "Renderer" específicas escritas en PHP.
*   **Alta Complejidad:** Para agregar una nueva plantilla, se requiere crear y registrar clases PHP separadas para cada componente (Header, Body, Footer) e integrarlas en el código del plugin.
