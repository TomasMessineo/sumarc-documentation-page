# Version 1.0

**Initial Release**
*   **Metadata Extraction**: Integration with the OJS API to obtain submission, publication, and context data.
*   **Front Generation**: Automatic construction of the `<front>` section conforming to JATS 1.2/1.3 and SciELO profiles.
*   **Preview**: Interface to review metadata changes before applying them.
*   **Workflow Integration**: Dedicated tab in the OJS Production stage.
