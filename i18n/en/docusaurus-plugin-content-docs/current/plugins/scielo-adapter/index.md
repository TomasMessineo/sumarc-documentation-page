# scieloAdapter

**scieloAdapter** is a specialization plugin for the SciELO ecosystem.

## Function
Adapts a standard JATS XML file (generated by SUMARC) to strictly comply with the specifications and validations required by the SciELO (Scientific Electronic Library Online) methodology.

## Features
*   Performs specific transformations on the XML structure.
*   Validates compliance with specific SciELO DTDs or schemas.
*   Ensures the document is accepted by the SciELO publishing system.

## Usage in Flow
It is an optional but crucial step if the publication destination is a journal indexed in SciELO. It is used on the already enriched XML before final delivery.
