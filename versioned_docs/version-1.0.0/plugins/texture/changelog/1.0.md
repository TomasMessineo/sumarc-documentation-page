# Versión 1.0

## Resumen
Versión base de Texture, correspondiente a la implementación estándar mantenida por el proyecto Substance/PKP.

## Características
- Editor WYSIWYG para JATS XML.
- Soporte para etiquetas estándar (párrafos, secciones, figuras, tablas).
- Interfaz amigable para la edición de referencias bibliográficas.
